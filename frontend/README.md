## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
## app
Add new pages of website to "app" folder. You must there is create a new folder of your component with file "page.tsx" in it. Don't put logic of website in it. 

## shared
All elements that can be used for several times add to "shared" folder.

## modules
All components add to "modules" folder. There are all logic of website.

## Form
Use "formik" for submitting form and "yup" for validation

## React-select
Use "react-select" for select inputs

## Tailwind
Use "tailwind" for styling
